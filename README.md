# Mathematica-code-for-computing-E-optimal-designs-for-second-order-Becker-s-mixture-models.
This repository contains Mathematica notebooks used to derive and verify E-optimal designs for second-order Becker’s mixture models under different model formulations. The files E-optBeckerH1andH3.nb and E-optBeckerH2.nb implement symbolic computations for the H1 and H3 models, and the H2  model, respectively. The code constructs the corresponding information matrices, computes their eigenvalues, and applies the E-optimality criterion by maximizing the smallest eigenvalue. It also solves the resulting systems of equations to obtain the optimal design weights and verifies the optimality conditions analytically.
